Bucket Sort

Bucket sort works by dividing the input elements into various groups or buckets. Once the buckets are created, we can sort these buckets individually by using any sorting algorithm, and the most common method is insertion sort.

How Does Bucket Sort Work?

The basic principal of bucket sort is as following:

  1. Creating a set of empty buckets. The number of buckets can be different in different cases.
  2. Assign the input array into corresponding bucket base on their range.
  3. Sort the elements within each buckets individually.
  4. Gather the sorted elements and create a single output array.

    Efficiency of Bucket Sort

    • The best case time complexity of bucket sort is O(n+k), where n is the number of the buckets and k is the bucket size.
    • The average case time complexity is O(n) when all elements are randomly distributed in the bucket.
    • The worst case time complexity is O(n^2), which there is only one bucket formed.

    Bucket sort is a efficient sorting method when sorting the uniformly distributed input elements.
